# 소설처럼 읽는 Node.js와 Express

Node.js + Express + TypeScript + MongoDB 웹 개발 교재의 챕터별 실습 코드입니다.

## 저장소 구조

각 브랜치는 해당 챕터까지의 코드가 누적 적용된 상태입니다.

## 브랜치 명명 규칙

```
part{N}/chapter-{M}
```

예: `part03/chapter-01` → PART 03 / 01장까지의 코드

## 실행 방법

```bash
npm install
npm run dev
```

Node.js 18 이상 필요. MongoDB 실행 필요 (PART 06부터).
